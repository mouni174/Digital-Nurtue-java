Box Model & Layout Control

Understanding Margin, Padding, Border, and Box Sizing

Box Model Components

The CSS Box Model consists of four layers:

  • Content: The actual content inside the box
  • Padding: Space inside the box, between content and border
  • Border: The boundary around the padding
  • Margin: Space outside the box, between border and adjacent elements

Margin Examples

Margin creates space OUTSIDE an element

Margin: 20px (all sides equal)

Margin: 10px top/bottom, 30px left/right

Margin: 5px top, 15px right, 25px bottom, 35px left

Margin: auto (centers the element)

Padding Examples

Padding creates space INSIDE an element, around the content

Small padding: 10px

Medium padding: 20px

Large padding: 30px
