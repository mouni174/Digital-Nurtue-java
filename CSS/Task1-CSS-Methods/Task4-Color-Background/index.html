HEX Color Values

#007BFF - Primary Blue

#28a745 - Success Green

#dc3545 - Danger Red

HEX values use six hexadecimal digits representing RGB values: #RRGGBB

RGB & RGBA Color Values

rgb(155, 89, 182) - Purple

rgba(52, 152, 219, 0.7) - Semi-transparent Blue

rgba(0, 0, 0, 0.5) - Semi-transparent Black

RGBA adds an Alpha channel for transparency (0 = fully transparent, 1 = fully opaque)
