Understanding CSS Selectors

This page demonstrates different CSS selector types and their applications.

Universal Selector (*)

The universal selector applies to ALL elements on the page.

We use it to reset default browser margins and padding.

Element Selectors

Target HTML elements directly by their tag name.

Example: p {}, h2 {}, div {}

ID Selector (#)

The mainHeader uses the ID selector. IDs are unique and have high specificity.

Example: #mainHeader {} targets only one element.

Class Selectors (.)

Classes are reusable and perfect for styling groups of elements.

Grouping Selectors

This heading and the following paragraphs are styled together

Grouping selectors with commas allows you to apply the same styles to multiple selectors.

Example: h2, p { color: #333; }
