Multi-Column Text Layout

The CSS3 multi-column layout feature allows you to display content in columns similar to a newspaper. This is useful for long-form text like articles, bulletins, and newsletters.

Column Balance Options

Balanced Columns (column-fill: balance)

This article demonstrates balanced column layout. The content is distributed evenly across columns, with the height of each column being approximately equal. This is the default behavior and provides a visually balanced appearance. The browser automatically calculates how much content goes in each column to minimize the height difference.

Balanced columns work well for editorial content where you want a polished, finished appearance. Long articles are distributed proportionally, preventing one column from being significantly taller than another.

Unbalanced Columns (column-fill: auto)

This article uses unbalanced column layout. Content fills columns sequentially from top to bottom. The first column fills completely before content moves to the next column. This creates a more flowing, left-to-right reading experience similar to how we read books.

Unbalanced columns are ideal for continuous reading experiences where you want to preserve the natural reading order and flow of content.

Column Gaps and Rules

Columns with Gap

This section demonstrates different column gap sizes. The column-gap property controls the space between columns. A larger gap improves readability by creating clear visual separation between columns. The default gap varies by browser but is typically around 1em.

Column gaps are important for visual hierarchy and preventing text from appearing too close together. They help guide the reader's eye across columns and improve overall readability of multi-column layouts.

Columns with Rules

Column rules are dividing lines between columns, created with the column-rule property. Rules can be customized with different styles (solid, dashed, dotted), widths, and colors. In this example, we're using a solid rule.

Rules provide visual separation between columns and can enhance the newspaper-like appearance of multi-column layouts. However, they should be used judiciously to avoid creating a cluttered appearance.

Spanning Elements Across Columns

This Heading Spans All Columns

Elements can span across multiple columns using the column-span property. Set column-span: all on an element to make it span across all columns. This is useful for headings, separators, and emphasis sections that should break out of the column layout.

When you use column-span: all, the element creates a break in the column flow. Content before the spanning element fills the columns normally, then the spanning element appears at full width, and content after it resumes the column layout.

This technique is particularly useful for article headlines, section breaks, and highlighted content that deserves to command the reader's full attention.

Responsive Multi-Column Design

Responsive Columns

Multi-column layout responds beautifully to different screen sizes. On desktop, this content might display in 3 columns. On tablets, it might display in 2 columns. On mobile devices, it naturally falls back to a single column for optimal readability.

Instead of specifying a fixed number of columns, you can use column-count: auto combined with column-width to let the browser determine the optimal number of columns based on available space. This provides a more flexible, responsive layout.

Resize your browser or use the device toolbar to see how this layout adapts to different screen sizes. The columns gracefully adjust to maintain readability across all devices.
